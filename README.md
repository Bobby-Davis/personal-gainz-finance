# PersonalGainz Finance App

A full-stack personal finance application for tracking bills, budgets, and monthly expenses. 

## Project Status
Early planning stage. Requirements and API design documentation are being created before backend and frontend development begins.

## Project Structure

Additional folders will be added for:
- `backend/` (Spring Boot API)
- `web/` (React web app)
- `mobile/` (React Native mobile app)

## Documentation
See the **docs** folder for:
- `requirements.md` – feature and MVP definition  
- `api-design.md` – (coming soon) backend API design

## Goals
- Backend: Spring Boot API with authentication and bill management  
- Web App: React interface for managing bills  
- Mobile App: React Native version for mobile users  
- Shared API for both clients  
- Clean, maintainable architecture following real-world engineering practices
